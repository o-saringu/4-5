プログラム復習コンテンツ

for構文

サンプルコードの結果

サンプルコード

　　 size(480, 120);

　　 for(int x = 60; x <= 420; x += 80 ){

　　 ellipse(x, 60, 60, 60);

　　　}

説明文

　for(初期値; 最大値; 増減値){ 繰り返し実行されるコード }

　for関数を使うことで繰り返しコードを実行することができる。　　

　初期値には繰り返しの初期の値を入れる。

　最大値には繰り返しの終点の値を入れる。

　増減値には繰り返すたびに実行される演算をいれる。

　

　例では、初期値ではx=60を設定して、xの値が420以下の場合xに80が足される。

　x=420を超えた場合forの繰り返しの処理が終わる。

　今回は計5回の繰り返し演算が行われている。
